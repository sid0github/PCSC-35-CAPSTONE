# AI-Powered 360¬∞ Feedback System for Government News

## Project Summary

This project is an advanced **AI-Powered 360¬∞ Feedback System** designed to analyze and categorize government-related news. It leverages state-of-the-art Machine Learning and Natural Language Processing (NLP) techniques to provide real-time insights into public sentiment and media coverage.

The system consists of a **FastAPI** backend and a **React (Vite)** frontend, offering a comprehensive dashboard for monitoring and analysis.

### Key Features
*   **Sentiment Analysis:** Automatically classifies news articles as Positive, Negative, or Neutral.
*   **Department Classification:** Categorizes news into 12 distinct government departments.
*   **Multi-language Support:** Supports analysis for 8+ Indian languages.
*   **Automated Alerts:** Sends email notifications for negative sentiment news.
*   **Content Extraction:** Capable of scraping URLs, performing OCR on PDFs, and analyzing raw text.

## Technology Stack

*   **Backend:** Python, FastAPI, PostgreSQL, SQLAlchemy
*   **Frontend:** React, Vite, TailwindCSS, Radix UI
*   **AI/ML:** PyTorch, Transformers, Scikit-learn, Tesseract OCR
